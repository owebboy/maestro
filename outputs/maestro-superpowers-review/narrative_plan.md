# Maestro + Superpowers Review

Audience: fellow coders evaluating agent workflow systems.

Objective: explain what Maestro and Superpowers each do, where they complement each other, what concrete review findings matter, and how to adopt them pragmatically.

Narrative arc:
1. Set the frame: these tools solve different layers of the workflow.
2. Show the system model: Maestro organizes work, Superpowers drives execution.
3. Present the engineering review: strengths first, then the concrete gaps.
4. End with practical adoption guidance instead of abstract praise.

Slides:
1. Title and framing.
2. The two-layer workflow model.
3. Review coverage snapshot.
4. Maestro strengths and verified findings.
5. Superpowers strengths and verified findings.
6. Adoption guidance for a real engineering team.
7. Close with the recommendation.

Visual system:
- Warm amber accent for Superpowers, teal accent for Maestro, slate neutrals.
- Bold title typography, high-contrast cards, editor-like dark surfaces.
- Geometric shapes and one native chart; no decorative photos needed.

Editability plan:
- All text stays in editable text boxes.
- The coverage comparison uses a native chart.
- Comparison diagrams use shapes and connectors only.
